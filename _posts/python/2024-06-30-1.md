## ❓Join()함수
: join 함수는 매개변수로 들어온 리스트에 있는 요소 하나하나를 합쳐서 하나의 문자열로 바꾸어 반환하는 함수입니다.  
<br />
'구분자'.join(리스트)
위와 같은 형태로 사용합니다.  
<br />
- 예시  
```python
a = ["abc", "ghi"]

for i in str_list:
  if ex not in i:
    a.append(i)
answer = "".join(a) # "abcghi"
```  

```python
a = ["abc", "ghi"]

for i in str_list:
  if ex not in i:
    a.append(i)
answer = "_".join(a) # "abc_ghi"
```
